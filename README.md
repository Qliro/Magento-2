# Qliro One Checkout for Magento 2
Qliro One for Magento 2 is an extension that integrates the Qliro One payment and checkout service into the Magento 2 e-commerce platform. Qliro One is a Nordic payment solution offering invoice, part payment, card payment, and direct bank payment options. The module is a fully functional implementation of a custom checkout that uses Qliro One functionality through its API.
